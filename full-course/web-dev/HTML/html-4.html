Tag Head, Tag Title, dan Tag Body

Tag Head

Tag Head berfungsi untuk menyimpan informasi dari sebuah halaman misalnya tag <title> (Judul dari sebuah Halaman) dan <meta> (pengkodean karakter UTF-8). Penulisan Tag head memiliki tag pembuka <head> dan tag penutup </head> apapun yang di tuliskan di dalam tag head tidak akan ditampilkan di halaman webiste yang kita buat.
Contoh penulisan tag head : 

<head>
  //isi dari informasi halaman, misalnya isi dari <title> dan <meta>
</head>

Tag Title

Tag title merupakan tag yang digunakan untuk menuliskan judul dokumen HTML. Hasil tag ini akan ditampilkan dalam window caption browser.
Contoh :

  • index.html
  • Output

Berkas HTML

<head>
   <title>Contoh dari Judul Website</title>
<head>

Hasil berkas HTML

Tag Body

Tag body merupakan bagian terdalam dari sebuah html, dan bagian ini yang akan di tampilkan dalam sebuah website.
Contoh penulisan tag body :

  • index.html
  • Output

Berkas HTML

<body>
    <h1> Contoh dari Judul </h1>
    <p> Contoh dari Paragpaph </p>
</body>
