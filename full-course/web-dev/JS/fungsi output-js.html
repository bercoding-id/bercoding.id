Fungsi Output javascript

Output adalah sebuah tampilan program yang pada umumnya untuk memperlihatkan hasil akhir. Output pada umumnya ditampilkan dalam bentuk teks dengan fungsi print()

Ada 4 cara menampilkan ouput pada javascript, diantaranya :

  1. Menggunakan Fungsi console.log();
  2. Menggunakan Fungsi alert();
  3. Menggunakan Fungsi document.write();
  4. Menggunakan Fungsi innerHTML.

Menggunakan Fungsi console.log()

Fungsi pada console.log() adalah fungsi untuk menampilkan teks ke console Javascript. Contoh penggunaannya seperti berikut : 

console.log("Hello World!");

fungsi console.log() pada umumnya digunakan untuk debugging. Dikatenakan setiap pesan error di JavaScript selalu ditampilkan di dalam Console.

Selain console.log(), terdap juga bebera fungsi yang bisa Anda gunakan untuk debugging seperti console.debug(), console.info(), console.error(), console.dir(), Dan masih banyak lainnya.

Menggunakan Fungsi alert()

Fungsi alert() adalah fungsi untuk menampilkan jendela dialog. Fungsi sebenarnya berada pada objek window.
Sebagai contoh seperti di bawah ini : 

  window.alert("Hello World!");

Bisa juga ditulis seperti dibawah ini : 

  alert("Hello World!");

Fungsi alert(), hanya bisa digunakan di dalam browser saja.
Supaya Anda mudah untuk memahami materi ini, dibawah ini kita akan menggunakan fungsi alert() dalam file HTML : 

  <html>
    <head>
     <meta charset="utf-8">
      <title>PenulisanFungsi Output JavaScript</title>
      <script>
        alert("Hi, Selamat datang di BeRCODING.ID");

        function sayHello(){
          alert("Hello!");
        }
      </script>
    </head>
    <body>
        <button onclick="sayHello()">Klik Dong </button>
    </body>
  </html>

Menggunakan Fungsi document.write()

Objek document ialah objek yang mewakili dokumen HTML di dalam Javascript.
Dalam objek document, terdapat fungsi write() untuk menulis sesuatu ke dokumen HTML.
Contoh : 

<html>
  <head>
    <meta charset="utf-8">
    <title>PenulisanFungsi Output JavaScript</title>
      <script>
          document.write("<h1>Hello Dunia!</h1>")
          document.write("<hr>")
          document.write("<p>Sela Berusaha Dan Berjuang</p>")
          document.write("Selamat Datang di <b>BeRCODING.ID</b>")
      </script>
  </head>
  <body>

  </body>
</html>

Menggunakan innerHTML

innerHTMLialah sebuah atribut di dalam (objek) elemen HTML yang berisikan sring HTML.
Dengan inner HTML, kita dapat menampilan output ke elemen yang lebih spesifik lagi.
Contohnya : 

<html>
  <head>
    <meta charset="utf-8">
    <title>PenulisanFungsi Output JavaScript</title>
  </head>
  <body>

    <h1>Belajar JavaScript</h1>
    <div id="output"></div>

      <script>
          // Membuat sebuah objek elemen
            var hasil = document.getElementById("output");

          // Menampilkan output ke elemen hasil
            hasil.innerHTML = "<p>I Love JavaScript</p>";
      </script>

  </body>
</html>
