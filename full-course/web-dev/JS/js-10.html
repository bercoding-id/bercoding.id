DOM API Untuk Manipulasi HTML

DOM merupakan singkatan dari Document Object Model.
Yang berarti, dokumen (HTML) yang dimodelkan dalam sebuah objek.
Objek dari dokumen ini menyediakan sekumpulan fungsi dan atribut/data yang bisa kita manfaatkan dalam membuat program JavaScript. Inilah yang disebut API (Application Programming Interface).
Dalam penggunakan DOM tidah hanya untuk dokumen HTML, tetapi DOM juga bisa digunakan untuk dokumen misalnya XML dan SVG. bt Cara Penggunaan objek DOM di JavaScript bernama document. Objek ini berisi segala hal yang kita butuhkan untuk memanipulasi HTML.
Jika kita coba ketikkan document pada console JavaScript, maka yang akan tampil adalah kode HTML.
Di dalam objek document, terdapat fungsi-fungsi dan atribut yang bisa kita pergunakan untuk memanipulasi dokument HTML. Kita bisa menggunakan fungsi document.write(), fungsinya untuk menuliskan sesuatu ke dokumen HTML.
Contoh : 

document.write("Hallo Dunia!");
document.write("<h2>Apa kabar semuanya</h2>");

Mengakses Elemen Tertentu dengan DOM

Objek document adalah model dari dokumen HTML. Objek ini berisi kumpulan fungsi dan atribut berupa objek dari elemen HTML.
Apabila Anda ingin mengakses elemen yang spresifik, terdapat beberapa fungsi yang bisa Anda gunakan, diantaranya :

  • getElementById() berfungsi untuk memilih elemen berdasarkan atribut dari id.
  • getElementByName() berfungsi untuk memilih elemen berdasarkan atribut dari name.
  • getElementByClassName() berfungsi untuk memilih elemen berdasarkan atribut dari class.
  • getElementByTagName() berfungsi untuk memilih elemen berdasarkan atribut dari nama tag.
  • getElementByTagNameNS() berfungsi untuk memilih elemen berdasarkan atribut dari nama tag.
  • querySelector() berfungsi untuk memilih elemen berdasarkan atribut dari query.
  • querySelectorAll() berfungsi untuk memilih elemen berdasarkan atribut dari query.
  • Dan lainnya.
