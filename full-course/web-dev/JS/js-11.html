FUNGSI

Fungsi adalah sub-program yang bisa digunakan kembali baik di dalam program itu sendiri, maupun di program yang lain. Fungsi di dalan JavaScript adalah sebuah objek. Karena memiliki properti, dalam fungsi di JavaScript ada 4 cara untuk membuatnya.

  1. Menggunakan cara biasa
  2. Menggunakan ekspresi
  3. Menggunakan tanda panah (=>)
  4. dan menggunakan Constructor.

Membuat Fungsi dengan Cara Biasa

Cara inilah yang paling sering digunakan, terutama pada pemula belajar JavaScript. 

function namaFungsi(){
  console.log("Hello Dunia!");
}

Membuat Fungsi dengan Ekspresi

Dalam membuat fungsi menggunakan ekspresi ini, kita menggunakan variabel, lalu diisi dengan fungsi. Fungsi ini sebenarnya adalah fungsi anonim (anonymous function) atau fungsi tanpa nama. 

var namaFungsi = function(){
  console.log("Hello Dunia!");
}

Menggunakan Fungsi dengan Tanda Panah

Dengan seiringnya waktu, cara inilah yang sering digunakan di kode JavaScript, karena cara ini jauh lebih sederhana. Akan tetapi sulit dipahami bagi pemula. Fungsi mulai muncul pada standar ES6. 

var namaFungsi = () => {
  console.log("Hallo Dunia!");
}

// Jika isi dari fungsi hanya satu baris Anda bisa menulis seperti ini
  var namaFungsi = () => console.log("Hallo Dunia!");

Menggunakan Fungsi dengan Kostruktor

Cara ini sebenarnya tidak direkomendasikan. Soalnya body fungsinya dibuat dalam bentuk string yang dapat mempengaruhi kinerja engine JavaScript. 

var namaFungsi = new Function('console.log("Hallo Dunia!");');

Bagi Anda yang baru mempelajari tentang JavaScript ini, kita rekomendarikan menggunakan cara yang pertama dulu, Nanti kalau Anda sudah terbiasa baru coba gunakan cara ke-2 dan ke-3.

Memanggil/Eksekusi Fungsi

Kita bisa memanggil fungsi di dalam kode JavaScript dengan menuliskan nama fungsinya seperti ini : 

namaFungsi()


Contohnya : 
// pembuatan Fungsi
  function sayHello(){
    console.log("Hallo Dunia!");
  }

// Memanggil fungsi
  sayHello() // maka hasilnya akan -> Hallo Dunia!

Selain dengan cara itu, kita juga bisa memanggil fungsi melalui atribut event, pada HTML. 
  <html>
    <head>
     <meta charset="utf-8">
      <title>Penulisan Fungsi</title>
      <script>
        // buat fungsi
        var sayHallo = () => alert("Hallo Kawan!");
      </script>
    </head>
    <body>
        <a href="#" onclick="sayHallo()">Klik Dong!</a>
    </body>
  </html>

Fungsi dengan Parameter

Parameter sendiri ialah variabel yang menyimpan nilai untuk diproses di dalam fungsi 

function bagi(a, b){
  hasilBagi = a / b;
  console.log("Hasil bagi a/b = " + hasilBagi);
}

Fungsi yang Mengembalikan Nilai

Supaya pengolahan nilai di dalam fungsi dapat digunakan untuk proses berikutnya, maka fungsi harus bisa mengembalikan nilai.
Untuk mengembalikan nilai pada fungsi, dengan menggunakan kata kunci return kemudian diikuti dengan nilai atau variabel yang akan dikembalika. 

function bagi(a,b){
  hasilBagi = a / b;
  return hasilBagi;
}

// Cara memanggil fungsi
  var nilai1 = 20;
  var nilai2 = 5;
  var hasil = bagi(nilai1, nilai2);

  console.log(hasil); // hasilnya akan -> 4
