Objek di JavaScript

Objek sebenarnya ialah sebuah variabel yang menyimpan nilai (properti) dan fungsi (method). Objek pada JavaScript, dapat dibuat dengan tanda kurung kurawal dengan isi berupa key dan value. Contoh pertama : 

var = {type:"Alfard", model:"600", color:"black"};

Contoh kedua: 
var car = {
  type:"Alfard",
  model:"600",
  color:"black"
};

Perbedaan Properti dan Method

Properti adalah ciri khas dari objek (variabel). Sedangkan method adalah perilaku dari objek (fungsi) tersebut.
Pembuatan method dapat dengan cara mengisi nilai (value) dengan sebuah fungsi. 

var car = {
  // properti
  type: "Ferrari",
  model: "Sport",
  color: "red",

  // method
  star: function(){
    console.log("ini method untuk start");
  },
  drive: function(){
    console.log("ini method untuk drive");
  },
  brake: function(){
    console.log("ini method untuk brake");
  },
  stop: function(){
    console.log("ini method untuk stop");
  }

};

Mengakses Properti dan Method Objek

Cara penggunaannya ialah dengan tanda titik atau dot (.), lalu setelah itu diikuti dengan nama properti atau method. Sebagai berikut : 

console.log(car.type);
console.log(car.color);

car.start();
car.drive();

Untuk mengakses properti, Anda cukup gunakan nama objek.properti. Sedangkan untuk method, kita harus menggunakan tanda kurun. Ini menyatakan kalau kita ingin mengeksekusi fungsi.

Menggunakan Keyword this
