Variabel dan Tipe Data

Dalam pembuatan variabel, pada umumnya digunakan di JavaScript dengan menggunakan kata kunci var lalu diikuti dengan nama variabel dan nilainya.
Contoh : 

var tittle ="Belajar Pemrograman di BERCODING.ID";

Pada contoh di atas, kita membuat variabel yang bernama title dengan nilai berupa teks (string) : "Belajar Pemrograman JavaScript".
Contoh Lagi Nih biar Anda semakin paham : 
var siteName = "BERCODING";
var url = "https://www.google.com";
var visitorCount = 2021;

Selain menggunakan kata kunci var kita juga bisa membuat variabel dengan kata kunci let atau tanpa awalan apapun. Seperti dibawah ini : 

// Membuat variabel dengan kata kunci let
  let price = 20000;

// variabel tanpa awalan apapun
  stock = 12

Perbedaan dari keduanya ialah, terletak pada jangkauan dan penggunaannya. Dan nilai dari variabel tersebut harus di isi, dikarenakan jika tidak diisi variabel yang Anda buat akan bernilai undefined (belum ditentukan).

Menampilkan isi Variabel

Unuk menampilkan isi dari variabel yang kita buat, kita bisa memanfaatkan dari fungsi-fungsi menampilkan output diantaranya :

  • Fungsi console.log() menampilkan sebuah output ke console JavaScript
  • Fungsi document.write() menampilkan output ke dokumen HTML
  • Fungsi alert() menampilkan output ke dalam jendela dialog
Setelah anda mengetahui dari setiap fungsinya, alangkah baiknya kita langsung praktik.
Sekarang Anda buatlah file bernama variabel-JavaScript.html, lalau anda bisa isikan dengan kode berikut ini : 
  <html lang="en">
    <head>
      <meta charset="utf-8">
      <title>Variabel dan Tipe Data</title>
        <script>
          // variabel
          var name = "BeRCODING";
          var year = 2021;
          var isActive = true;
          var url = "https://www.BeRCODING.com";

          // menampilan dari variabel ke jendela dialog (alert)
          alert("Selamat datang di " + name);

          // menampilkan dari variabel ke dalam HTML
          document.write("Tempat Belajar : " + name + "<br>");
          document.write("Tahun Berdiri : " + year + "<br>");
          document.write("Status Aktif : "+ isActive + "<br>");
          document.write("Alamat Url : " + url + "<br>");
        </script>
    </head>
    <body>

    </body>
  </html>

Setelah Anda sudah menuliskan kode nya, coba Anda buka dengan browser dan perhatikanlah hasilnya.


Arti dari simbo (+) pada contoh code di atas adalah untuk menggabungkan, bukan menjumlahkan.

Mengisi Ulang Variabel

Variabel yang bersifat mutable, tidak membatasi kita untuk merubah ulang nilai dari variabel sebelumnya, dikarenakan dengan memiliki sifat mutabel variabel bisa merubah ulang nilai sebelumnya.
Contoh 

// Nilai awal variabel
  var age = 21;

// Nilai bari pada variabel sebelumnya
  age = 25;

Dalam merubah nilai variabel sebelumnya, kita tidak memerlukan kata kunci var dikarenakan kata kunci itu hanya dibutuhkan di saat kita membuat variabel saja. Sedangkan untuk merubah nilai variabel sebelumnya kita cukup tulis seprti di atas.
Apabila kita menggunakan kata kunci var, berarti kita sedang membuat variabel baru, bukan merubah nilai variabel sebelumnya.

Menghapus Variabel

Menghapus variabel dalam JavaScript sangat jarang sekali dilakukan. Tetapi, untuk program yang membutuhkan ketelitian dalam alokasi memori, penghapusan variabel perlu dilakukan agar penggunaan memori lebih optimal lagi.
Menghaspus variabel dapat dilakukan dengan kata kunci delete.
Contoh : 

bookTittle = "Pemrograman JavaScript";
delete bookTittle;

Ketika kita menggunakan kata kunci delete, maka variabel bookTittle akan menghilang dari penyimpanan memori
Penghapusan variabel hanya bisa dilakukan pada variabel yang dibuat tanpa awalan. Sedangkan yang dibuat dengan kata kunci var dan let akan dihapus otomatis.

Mengenal Tipe Data

Tipe Data adalah jenis-jenis data yang bisa kita simpan ke dalam variabel.
Ada beberapa tipe data dalam Pemrograman JavaScript, diantaranya :

  • String (teks)
  • Integer atau Number (bilangan bulat)
  • Float (bilangan pecahan)
  • Boolean
  • Objct
JavaScript memiliki sifat dynamic typung, yang artinya kita tidak harus menuliskan tipe data pada saat proses pembuatan variabel seperti pada baha Pemrograman C, C++, Dan lainnya. yang bersifat static typing.
Contoh : 
var name = "Ramdani";
var age = 21;
var single = true;

JavaScriptakan secara otomatis mengenali tipe data yang kita berikan pada variabel tersebut.
Pada contoh di atas akan memiliki tipe data seperti dibawah ini :
  • name memiliki data string
  • age memiliki tipe data Integer/number
  • dan single akan memiliki tipe data boolean.
Hal ini bisa juga kita cek dengan menggunakan kata kunci tyeof.
Contoh : 
typeof name;
typeof age;
typeof single;

// Untuk tipe data teks atau string cara penulisannya selalu diapit dengan tanda petik
  var author = "BeRCODING.ID";

// sedangkan tipe data number dan booleadn tidak diampit oleh apapun
var single = true; // < tipe data boolean hanya bernilai true dan false, tidak ada lagi.
var age = 21;

Aturan Dalam Penulisan Nama variabel di JavaScript

Dalam penulisan variabel, ada beberapa aturan npenulisan, diantaranya :

  • Penamaan variabel tidak boleh menggunakan anga di depannya.
    Contoh : 
    // penulisan yang salah
  var 21nama ="BeRCODING";

// penulisan yang benar
  var nama21 = "BeRCODING";
  • Penamaan variabel boleh menggunakan awal underscore.
    Contoh : 
    var _nama = "BeRCODING";
  • Penamaan variabel dianjurkan menggunakan camelCase apabila terdiri dari dua suku kata.
    Contoh : 
    var fullName = "BeRCODING.ID";
