Percabangan

Percabangan if

Percabangan if merupakan percabangan yang memiliki satu blok pilihan saja , di saat kondisi bernilai benar.
Penulisannya sebagai berikut : 

    <!DOCTYPE html>
    <html>
    <head>
      <title>Operator</title>
    </head>
    <body>
      <script>
        var totalBelanja = prompt("Total Belanja?", 0);

        if(totalBelanja > 100000){
          document.write("<h2>Selamat Anda dapat hadiah!!!</h2>");
        }
        document.write("<p>Terimakasi sudah mampir disini</p>");
      </script>
    </body>
    </html>

Percabangan if/else

Percabangan if/else merupakan percabangan yang memiliki dua blok pilihan.
Pilihan pertama untuk kondisi benar, dan pilihan kedua untuk kondisi sala (else).
Penulisannya sebagai berikut : 

  <!DOCTYPE html>
  <html>
  <head>
    <title>Operator</title>
  </head>
  <body>
    <script>
      var password = prompt("Password : ");

      if(password == "gedung"){
        document.write("<h2>Selamat Datang!!!</h2>");
      }else{
        document.write("<p>Pasword yang anda masukan salah,</p>");
      }
      document.write("<p>Terimakasi sudah mampir disini</p>");

    </script>
  </body>
  </html>

Percabangan if/else/if

Percabangan if/else/if merupakan percabangan yang memiliki lebih dari dua blok pilihan.
Ini adalah blok untuk percabangan if/else/if. Kita bisa menambahkan berapapun blok yang kita inginkan. 

  <!DOCTYPE html>
  <html>
  <head>
    <title>Operator</title>
  </head>
  <body>
    <script>
      var nilai = prompt("Inputkan nilai akhir:");
      var grade = "";
      if(nilai >= 90) grade = "A"
      else if(nilai >= 80) grade = "B+"
      else if(nilai >= 70) grade = "B"
      else if(nilai >= 60) grade = "C+"
      else if(nilai >= 50) grade = "C"
      else if(nilai >= 40) grade = "D"
      else grade = "F";

      document.write(`<p>Grade anda : ${grade}</p>`);

    </script>
  </body>
  </html>

Percabangan switch/case

Percabangan switch/case adalah bentuk lain dari percabangan if/else/if.
Struktur percabangan switch/case, seperti berikut : 

switch(variabel){
  case<value>:
    // blok kode
    break;
    case<value>:
      // blok kode
      break;
    default:
      // blok kode
}

Dalam pembuatan blok kode (case) tidak ada batasannya, kita bisa membuat sebanyak yang diinginkan di dalam blok switch.
<value>, kita bisa isi dengan nilai yang nanti akan dibandingkan dengan variabel.
setiap case harus diakhiri dengan breakl. Khusus untuk default, tidak perlu diakhiri dengan break karena dia terletak di bagian akhir.
Pemberian break bertujuan agar program berhenti mengecek case berikutnya saat sebuah case terpenuhi.
Contohnya : 
<!DOCTYPE html>
<html>
<head>
  <title>Operator</title>
</head>
<body>
  <script>
    var answer = prompt("Kamu Beruntung! Silahkan pilih hadiamu dengan memasukan angka 1 sampai 5");
    var hadiah = "";

    switch(answer){
      case "1":
        hadiah = "Roko";
        break;
      case "2":
        hadiah = "Mie";
        break;
      case "3":
        hadiah = "Sticker";
        break;
      case "4":
        hadiah = "Es Cream";
        break;
     case "5":
        hadiah = "Cash Rp. 1.000.000";
        break;
     default:
        document.write("<p>Woow! Anda salah memilih</p>");
    }
    if(hadiah == ""){
      document.write("<p>Kamu gagal mendapatkan hadiah</p>");
    }else {
      document.write("<h2>Selamat Kamu telah mendapatkan hadiah " + hadiah + "</h2>");
    }

  </script>
</body>
</html>
