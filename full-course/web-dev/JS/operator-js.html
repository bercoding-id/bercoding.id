Operator

Operator ialah sebuah simbol yang digunakan untuk melakukan operasi tertentu pada nilai dan variabel.
Dalam operator, di pemrograman dibagi menjadi 6 jenis, diantaranya :

  1. Operator aritmmatika
  2. Operator Penugasan (Assigment)
  3. Operator relasi atau perbandingan
  4. Operator Logika
  5. Operator Bitwise
  6. Operator Ternary
Operator adalah hal yang wajib ada di setiap pemrograman. Ke 6 jenis operator tersebut harus Anda pahami betul-betul.

Operator Aritmatika

Operator aritmatika merupakan operator untuk melakukan operasi aritmatika seperti penjumlahan, pengurangan, pembagian, perkalian, Dan lainnya.
Operator aritmatika terdiri dari :

Nama Operator Simbol
Penjumlahan +
Pengurangan -
Perkalian *
Pemangkatan **
Pembagian /
Sisa Bagi %

Untuk melakukan operasi perkalian, kita menggunakan simbol asterik *. Jangan gunakan X, karena simbol X bukan termasuk dalam pemrograman.
Dan untuk pemangkatan kita menggunakan asterik ganda **.
Untuk pembagian, kiga gunakan simbol garis miring / Contoh : 
var a = 4;
var b = 2;

// contoh penggunaan operator penjumlahan
  var c = a + b;
  console.log(c);

Anda juga bisa menggunakannya dengan fungsi dari document.write(), Sebagai contoh : 
  <!DOCTYPE html>
  <html>
    <head>
      <title>Operator</title>
    </head>
    <body>
      <script>
        var a = 10;
        var b = 2;
        var c = 0;

        // Operator Pengurangan
          c = a - b;
          document.write('${a} - ${b} = ${c}<br>');

        // Operator Perkalian
          c = a * b;
          document.write('${a} * ${b} = ${c}<br>');

        // Operator Pemangkatan
          c = a ** b;
          document.write('${a} ** ${b} = ${c}<br>');

        // Operator Pembagian
          c = a / b;
          document.write('${a} / ${b} = ${c}<br>');

        // Modulo / Sisa Bagi
          c = a % b;
          document.write('${a} % ${b} = ${c}<br>');
      </script>
    </body>
  </html>

Coba Anda perhatikan operator modulo/sisa bagi (%) dan operator (+). Operator modulo adalah operaotr untuk menghitung sisa bagi.
Misalkan 3 dibagi 2, maka sisanya adalah 1. 
3 % 2 = 1

Operator Penggabung Teks

Dalam JavaScript, jika kita menggunakan operator (+) terhadap tipe data string atau teks, maka yang akan terjadi adalah penggabungan. Bukan penjumlahan.
Contoh : 

var a = "5" + "2";

Maka hasilnya akan seperti ini : 
52

Kenapa tidak (7)? Karena kedua nilai, Merupakan string, Anda bisa memerhatikan penulisannya, dia diampit oleh tanda petik.
Untuk operasi yang lainnya, silahkan Anda dicoba melalui console

Operator Penugasan pada JavaScript

Operator penugasan adalah operator yang digunakan untuk memberikan tugas kepada variabel, Pada umumnya digunakan untuk mengisi variabel.
Contoh dari penulisannya : 

var a = 21;

Kita baru saja memberikan tugas ke variabel (a) untuk menyimpan nilai (21).
Operator penugasan terdiri dari :
Nama Operator Simbol
Pengisian Nilai =
Penambahan dan Pengisian +=
Pengurangan dan Pengisian -=
Perkalian dan Pengisian *=
Pemangkatan dan Pengisiaan **=
Pembagian dan Pengisian /=
Sisa Bagi dan Pengisian %=
Operator penugasan sama seperti operator aritmatika, bisa juga digunakan untuk melakukan operasi aritmatika.

Operator Perbandingan pada JavaScript

Operator relasi atau perbandingan adalah operator yang sering digunakan untuk membandingkan duan nilai.
Operator perbandingan akan menghasilkan sebuah nilai boolean true dan false.
Operator perbandingan terdiri dari :

Nama Operator Simbol
Lebih Besar >
Lebih Kecil <
Tidak Sama Dengan != atau !==
Sama Dengan == atau ===
Lebih Kecil Sama Dengan <=
Lebih Besar Sama Dengan >=
Contoh Penulisannya sebagai berikut : 
<!DOCTYPE html>
<html>
  <head>
    <title>Operator</title>
  </head>
  <body>
    <script>
    var me = 21;
    var you = 19;

    // sama dengan
    var hasil = me == you;
    document.write(`${me} == ${you} = ${hasil}<br>`);

    // lebih besar
    var hasil = me > you;
    document.write(`${me} > ${you} = ${hasil}<br>`);

    // lebih besar samadengan
    var hasil = me >= you;
    document.write(`${me} >= ${you} = ${hasil}<br>`);

    // lebih kecil
    var hasil = me < you;
    document.write(`${me} < ${you} = ${hasil}<br>`);

    // lebih kecil samadengan
    var hasil =me <= you;
    document.write(`${me} <= ${you} = ${hasil}<br>`);

    // tidak samadengan
    var hasil = me != you;
    document.write(`${me} != ${you} = ${hasil}<br>`);
    </script>
  </body>
</html>

Operator Logika pada JavaScript

Operator logika pada umumnya digunakan untuk melakukan operasi terhadap dua nilai boolean.
Operatornya terdiri dari :

Nama Operator Simbol
Logika AND &&
Logika OR ||
Negasi atau Kebalika !
Supaya Anda mudah untuk memahaminya ini contoh dari penulisannya : 
<!DOCTYPE html>
<html>
  <head>
    <title>Operator</title>
  </head>
  <body>
    <script>
    var me = 21;
    var you = 19;

    var benar = me > you;
    var salah = me < you;

    // Operator && (AND)
    var hasil = benar && salah;
    document.write(`${benar} && ${salah} = ${hasil}<br>`);

    // Operator || (OR)
    var hasil = benar || salah;
    document.write(`${benar} || ${salah} = ${hasil}<br>`);

    // Operator ! (NOT)
    var hasil = !benar;
    document.write(`!${benar} = ${hasil}<br>`);

    </script>
  </body>
</html>

Operator Bitwise pada JavaScript

Operator bitwise merupakan operator yang digunakan untuk operasi berdasarkan bit (biner).
Operator bitwise terdiri dari :

Nama Simbol di Java
AND &
OR |
XOR ^
Negasi/Kebalikan ~
Operator ini berlaku untuk tipe data int, long, short, char, byte.
Operator ini akan menghitung dari bit-ke-bit.
Misalnya, Anda punya variabel a = 60 dan b = 13.
Bila dibuat dalam bentuk biner, akan menjadi seperti ini : 
a = 00111100
b = 00001101

Kemudian, dilakukan operasi bitwise.
Operasi AND 
a = 00111100
b = 00001101
a & b = 00001100

Operasi OR 
a = 00111100
b = 00001101
a | b = 00111101

Operasi XOR 
a = 00111100
b = 00001101
a ^ b = 00110001

Operasi NOT (Negasi/Kebalika) 
a = 00111100
~a = 11000011

Dalam penulisannya hampir sama dengan operator Logika. Tetapi dalam penggunaannya Bitwise digunakan untuk biner. 
<!DOCTYPE html>
<html>
<head>
  <title>Operator</title>
</head>
<body>
  <script>
        var x = 4;
        var y = 3;

        // operator bitwise and
        var hasil = x & y;
        document.write(`${x} & ${y} = ${hasil}<br>`);

        // operator bitwise or
        var hasil = x | y;
        document.write(`${x} | ${y} = ${hasil}<br>`);

        // operator bitwise xor
        var hasil = x ^ y;
        document.write(`${x} ^ ${y} = ${hasil}<br>`);

        // operator bitwise negasi
        var hasil = ~x;
        document.write(`~${x} = ${hasil}<br>`);

  </script>
</body>
</html>
