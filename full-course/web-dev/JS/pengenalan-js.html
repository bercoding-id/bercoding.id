Pengenalan javascript

JavaScript adalah bahasa pemrograman yang wajib kamu pelajari jika ingin mendalami duni web Development. Saat ini JavaScript tidak hanya digunakan di sisi client (browser) saja. Tetapi JavaScript juga digunakan pada server, console, program desktop, mobile, IoT, game, dan masih banyak lainnya. Hal ini membuat bahasa JavaScript semakin populer.

Apa Itu JavaScript?

JavaScript adalah bahasa pemrograman yang awalnya dirancang untuk berjalan di atas browser. Namun, seiring perkembangan zaman, javascript tidak hanya berjalan di atas browser saja. Tetapi JavaScript juga dapat digunakan pada sisi server, Game, IoT, Destop, Dan lainnya.
JavaScript awalnya bernama Mocha, lalu berubah menjadi LiveScript saat browser Netscape Navigator 2.0 rilis versi beta (September 1995). Namun, setelah itu dinamai ulang menjadi javascript.
Pada tahun 1996, Netscape mengirimkan standarisasi ECMA-262 ke Ecma International. Sehinga lahirlah standarisasi kode JavaScript bernama ECMAScript atau ES. Saat ini ECMAScript sudah mencapai versi 8 (ES8).

  2. Teks Editor (Rekomendasi VS Code)

Mengenal Console JavaScript

Sering kita dengar kalau belajar JavaScript iu susah, dikarenakan saat melihat hasilnya di web browser, pesan error-nya tidak tampil. Pendapat itu tidak benar. karena kita bisa melihatnya melalui console.
Console JavaScript dapat kita buka melalui Inspect Element -> Console.

Di dalam Console, kita bisa menulis fungsi atau kode-kode javascript dan hasilnya akan langsung ditampilka.
Misalnya, mari kita coba dengan code berikut ini : 
console.log("Hi Brother");
alert("Saya sedang belajar JavaScript");
