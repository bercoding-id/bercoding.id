Penulisan javascript

JavaScript memiliki 3 gaya dalam penulisannya diantaranya :

  1. Embed (Kode JavaScript ditempel langsung pada HTML)
  2. Inline (Kode JavaScript ditulis pada atribut HTML)
  3. Eksternal (Kode JavaScript ditulis terpisah dengan file HTML)

Menulis JavaScript pada Tag (Embed)

Dalam penulisan kode JavaScript menggunakan tag <script> adalah cara yang sudah sering para web Development gunakan. Tag <script> dapat kita buat di dalam tag <head> maupun <body>.
Contoh dalam penulisannya, Anda bisa lihat seperti dibawah ini : 

<html>
  <head>
   <meta charset="utf-8">
    <title>Penulisan JavaScript</title>
    <script>
      console.log("Hi, Ini kode JavaScript");
    </script>
  </head>
  <body>
      document.write("Javascript itu keren!");
  </body>
</html>

Hasilnya :


Mungkin anda bingung untuk menentukan mana yang lebih baik dari 2 cara penulisan tersebut. Kedua cara ini memang tidak di salahkan tetapi penulisan di akhir <body> akan lebih meningkatkan kecepatan displat atau tampilannya konten di web.
Karena, jika kita tulis di dalam <head>, script tersebut akan dieksekusi terlebih dahulu dikarenakan dalam HTML program itu akan membaca dari atas ke bawah. Jadi, disarankan dalam menulis kode javascript di dalam <body>. tepatnya sebelum tag penutup </body>

Menulis JavaScript pada File Eksternal

Cara kedua, kita bisa menulis Javascript di file yang terpisah dengan HTML. Biasanya digunakan jika kita tidak ingin Javascript bercampur dengan kode HTML (kesukaan masing-masing).
Caranya, buatlah sebuah file yang berekstensi .js, dan di simpan dalam satu folder yang sama. File ini kita isi denga kode javascript
Dan ini contoh isi dari file tersebut : 

document.write("Kode Javascript penulisan Eksternal")
lalu isi file HTML dengan code dibawah ini : 
<html>
  <head>
      <meta charset="utf-8">
      <title>Penulisan Eksternal</title>
  </head>
  <body>

      <script src="kode-script.js"></script>

  </body>
</html>

Maka hasilnya akan seprti dibawah ini :


Pada file HTML, kita tetap menggunakan tag <script>. Akan tetapi, tag ini tidak kita isi dengan kode JavaScript. Karna kita hanya membuat sebuah tautan yang menghubungkan antara file HTML dan File JavaScript, dengan menggunakan atribut src untuk menjalankan kode javascript yang ada di file eksternal. Walaupun tidak punya isi tag <script> harus tetap di tutup dengan tap penutup </script>.
Dan jika file javascript-nya berada di folder yang berbeda dengan file HTML, kita bisa menambahkan sesuai alamat path foldernya. Misal struktur foldernya seperti ini :
Jika file kode-script.js berada di dalam folder js, maka Anda harus menulisnya : 


<script src="js/kode-script.js"></script>

Menulis JavaScript Pada Javascript (Inline)

Untuk penulisan JavaScript pada atribut pada umumnya dilakukan pada atribut event.
Contoh : 

<button onclick="alert('Selamat Anda Berhasil !!!')">Coba Di Klik</button>

Hasil dari code di atas akan seperti di bawah ini :

Penulisan javascript tidak hanya bisa di tulis di atribut onclick saja, atribut onclick adalah salah satu event yang ada di HTML, kode JavaScript bisa ditulis di semua atribut event. Selalu ingat bahwa penulisan JavaScript harus pada atribut event! bukan atribut biasa
Penulisan atribut event di HTML diawali dengan on, misalnya untuk event click, maka pada atribut HTML ditulis onclick.
Contoh, supaya Anda bisa mencintai materi ini : 

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Penulisan Javascript di Atriibut></title>
</head>

<body>
    <textarea onkeyup="document.getElementById('counter').innerText = this.value.length" cols="30" rows="10"></textarea>
    <p>Panjang: <span id="counter">0</span> karakter</p>
</body>

</html>



Pada contoh ini, kita menggunkan even keyup, maka ditulis onkeyup. Event ini terjadi saat kita melepas tombol di keyboard. Jadi nantinya saat kita mengetik pada elen &l;textarea>, maka kode JavaScript yang ada di onkeyup akan dieksekusi. Kode JavaScript tersebut berfungsi untuk menampilkan jumlah karakter yang di ketik di dalam textarea.
