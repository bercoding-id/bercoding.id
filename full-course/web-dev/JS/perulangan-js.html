Perulangan dapat membantu kita untuk mengeksekusi kode yang berulang-ulang berapapun yang kita ingingkan.

Secara umum, perulangan dibagi menjadi dua bagian, yaitu : counted loop dan uncounted loop.

  • Counted Loop Merupakan perulangan yang jelas dan sudah tentu banyak perulangannya
  • Sedangkan Uncounted Loop, merupakan perulangan yang tidak jelas berapa kali ia harus mengulanginya
Perulangan yang termasuk ke dalam Counted Loop :
  1. Perulangan For
  2. Perulangan Foreach
  3. Perulangan Repeat
Perulangan yang termasuk ke dalam Uncounted Loop :
  1. Perulangan While
  2. Perulangan Do/While

Perulangan Menggunakan For

Dikarenakan perulangan for sudah jelas berapa kali dia akan mengulang, jadi dia termasuk ke dalam Counted Loop.
Struktur penulisannya seperti berikut : 

for(let i = 0; i < 10; i++){
  document.write("<p>Perulangan ke- " + i + "</p>")
}

Yang perlu Anda sangat perhatikan ialah, kondisi yang ada di dalam kurung setelah kata for.

Kondisi ini yang akan menentukan berapa kali akan mengulanginya :
  • Hitungan akan dimulai dari 0 (i = 0)
  • Hitungannya sampai berapa? Sampai i < 10
  • Di setiap perulangan i akan bertambah +1 (i++)
Variabel i pada perulangan for berfungsi untuk menyimpan nilai hitungan.
Jadi setiap perulangan dilakukan niali i akan selalu berambah satu. Karena kita menentukannya di bagian i++ Hasil output dari code yang kita tuliskan barusan, sebagai berikut :



Tidak hanya untuk menghitung maju, denga for, kita juga bisa menghitung mundur
Sebagai Berikut : 
for(counter = 15; counter > 0; counter--){
  document.write("<p>Perulangan ke " +counter+"</p>");
}

Lah ko kenapa hitungannya tidak sampai (0)?
Nah, karena kondisi yang kita berikan kepada counter > 0. Maka apabila counter bernilai 0, maka kondisi ini akan menjadi false.
Kecuali kita menggunakan operator lebih besar sama dengan (>=), maka jika counter bernilai 0, kondisi akan menjadi true.

Perulangan Menggunakan While

Perulagan while merupakan perulangan yang termasuk dalam perulangan Uncounted Loop.
Perulangan while juga dapat menjadi perulangan yang counted loop dengan memberikan counter di dalamnya.
Sebagai contoh berikut ini : 

var ulangi = confirm("Apakah anda ingin mengulanginya?");
var counter = 0;

  while(ulangi){
    counter++;
    ulangi = confirm("Apakah anda ingin mengulanginya?");
}

  document.write("Perulangan sudah dilakukan sebanyak " + counter + "Kali");
Hasilnya akan seperti, berikut ini :


Di code yang barusan kita buta, perulangan selama variabel ulangi bernilai true.
Di code itu kita menggunakan juga fungsi dari confirm() untuk menampilkan dialog konfirmasi.
Selama Anda memilik OK pada konfirmasi, maka variabel ulangi akan terus menerus bernilai true. Namun, jika Anda memilih Cancel, maka variabel ulangi akan berubah nilainya menjadi false.
Saat variabel ulangi bernilai false, disitulah perulangan akan berhenti.

Perulangan Menggunakan Do/While

Perulangan do/while akan melakukan perulangan sebanyak 1 kali terlebih dahulu, lalau mengecek kondisi yang ada di dalam kurung while.
Struktur penulisannya, sebagai berikut : 

do {
  // blok kode yang akan diulangi
} while (<kondsi>);

Perbedaan antara while dan do/while ialah :
Perulangan Do/While akan mengecek kondisi di belakang (sesudah mengulang),
sedangkan while akan mengecek di depan atau diawalan (sebelum mengulan).

Perulangan Menggunakan Foreach

Perulangan Foreach pada umumnya digunakan untuk mencetak item di dalam array.
Perulangan ini termasuk ke dalam perulangan counted loop, karena jumlah perulangannya akan ditentukan oleh panjang nya jumlah dari array.
Ada dua cara untuk menggunakan perulangan foreach, diantaranya :

  1. Menggunakan for dengan operator in
  2. Menggunakan method forEach()
Sebagai contoh :

Berikut ini bentuk perulangan "foreach" tanpa menggunakan operator in : 
var languages = ["JavaScript", "CSS", "HTML", "Python"];

for(i = 0; i < languages.length; i++){
  document.write(i+ ". "+ languages[i] + "<br>");
}


Perulangan ini dapat dibuat lebih sederhana lagi dengan menggunakan operator in, sebagai berikut : 

var languages = ["JavaScript", "CSS", "HTML", "Python"];

for(i in languages){
  document.write(i+". " + languages[i] + "<br>");
}

Cara kedua dalam membuat perulangan foreach dengan menggunakan method forEach() dari array.
Contoh : 

// nilai array
  var days = ["Senin", "Selasa", "Rabu", "Kamis", "Jum'at", "Sabtu", "Minggu" ];

// Lalu kita tampilkan semua hari
// dengan menggunakan method forEach
days.forEach(function(day){
  document.write("<p>" +day + "</p>");
});
