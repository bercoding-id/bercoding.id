Array PHP

Secara sederhana Array adalah variabel yang bisa diisi oleh banyak data sekaligus, dan didalam array bisa terdiri dari beberapa kumpulan tipe data. Semisal dalam kelompok buah terdiri dari 5 nama buah, lalu kita menyimpan 5 nama buah tersebut kedalam variabel, maka penulisannya seperti berikut : 

<?php
  $buah = array(
      0 => 'Semangka',
      1 => 'Mangga',
      2 => 'Kedongdong',
      3 => 'Jeruk',
      4 => 'Sirsak'
  );
  ?>
Cukup mudah bukan?, Anda cukup menuliskan variabel $buah, lalu didalamnya buat array yang berisi dari nama-nama buah. Untuk kcara pemanggilannya, semisal kita ingin menampilkan nama buah "Kedongdong", caranya jika kita menggunakan array tinggal tuliskan echo $buah['2']; yang berarti kita memanggil data index ke 2 pada array $buah. 
<?php
  $buah = array(
    0 => 'Semangka',
    1 => 'Mangga',
    2 => 'Kedongdong',
    3 => 'Jeruk',
    4 => 'Sirsak'
  );

  echo $buah['2']; // Mengambil lalu menampilkan data index ke 2 pada array $buah
?>

Penulisan Array

Dalam penulisan array ada 2 cara, diantaranya :

  • Menggunakan array()
  • Menggunakan tanda kurung siku []

Jenis-jenis Array

Array memiliki 3 jenis dalam PHP, diantaranya :

  • Indexed Array
  • Associative Array
  • Multidimensional Array
Dibawah ini adalah sebuah ilustrasi dari setiap jenis array dalam PHP :

Perbedaan dari ketiga jenis tersebut, ialah :
  1. Indexed array menggunakan index berupa angka, yang secara default diawali dengan index 0
  2. Associative array menggunakan index yang bukan angka seperti Indexed Array
  3. Multidimensional array1 memungkinkan kita untuk membuat array didalam array

Indexed Array

Indexes array ialah, array yang memiliki index atau key berupa angka yang berurutan.
Contohnya: 

<?php
  $mahasiswa = ['Rudi', 'Rahmat', 'Ananda'];
  echo $mahasiswa['2']; // Menampilkan Ananda

  $mobil = [1 => 'Ferrari', 'Mercedes', 'BMW'];
  echo $mobil['2']; // Menampilkan Mercede

  $buah = ['Apel', 'Sirsak', 15=> 'Mangga'];
  echo $buah['15']; // Menampilkan Mangga
?>
Penjelasan :
  • Pada array $mahasiswa index array dimulai dari angka 0, jadi index ['0'] = Rudi, index ['1'] = Rahmat index ['2'] = Ananda, maka ketika kita menuliskan $mahasiswa['2'] yang tampil adalah Ananda
  • Pada array $mobil index array dimulai dari angka 1, dikarenakan untuk apel kita tuliskan index 1, maka ketika kita menuliskan $mobil['2'] yang akan tampil adalah Mercedes, jadi anda bisa mendeklarasikan nilai awal dari index tersebut.
  • Pada array $buah kita menuliskan index 15 adalah Mangga, maka saat kita menuliskan $buah['15'] yang akan tampil adalah Mangga, jadi anda bisa mendeklarasikan nomor index secara manual.

Associative Array

Associative Array memiliki kesamaan dengan indexed array, yang menjadi pembedanya hanyalah Associative array ini indexnya tidak berupa angka, sebagai contoh : 

<?php
  $mahasiswa = ['id' => 0021, 'nama' => 'Ramdani', 'kelas' => '12', 'jenis_kelamin' => 'Pria', 'agama' => 'islam'];
  echo $mahasiswa['nama']; // Yang akan tampil Ramdani
?>
Penjelasan :
  • Pada array $mahasiswa dengan jenis Associative array kita menuliskan beberapa key, seperi id, nama, kelas, jenis_kelamin, dan agama, serta mengisikan nilai di masing-masing key, maka ketika kita menusliskan $mahasiswa['nama'] yang akan tampil adalah Ramdani.

Multidimensional Array

Multidimensional Array sederhananya adalah Anda bisa membuat array di dalam array, sebagai contoh : 

<?php
    $siswa = [
        'Indah'=>[
                  'id'=>1,
                  'nama'=>'Indah',
                  'kelas'=>'12',
                  'jenis_kelamin'=>'Perempuan',
                  'agama'=>'islam'
         ],
         'Ramdani'=>[
                  'id'=>2,
                  'nama'=>'Ramdani',
                  'kelas'=>'10',
                  'jenis_kelamin'=>'pria',
                  'agama'=>'kristen'
          ],
     ];

      echo $siswa['Ramdani']['jenis_kelamin']; //menampilkan pria
?>
Penjelasan :
  • Kita membuat array $siswa yang didalamnya ada 2 index yaitu ['0'] = 'Indah', dan ['1'] = 'Ramdani'.
  • Pada masing-masing value index kita buat array lagi didalamnya
  • Sehingga jika ingin menampilkan jenis kelamin dari Ramdani, Anda bisa menampilkannya dengan menuliskan $siswa['Ramdani']['jenis_kelamin'], semisalnya kita ingin menampilkan agama dari Indah maka tuliskan $siswa['Indah']['agama'];
