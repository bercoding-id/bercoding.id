Pengenalan PHP

PHP ialah singkatan dari Hypertext Prepocessor dan merupakan salah satu bahasa pemrograman yang di design khusus untuk web development atau pengembangan web. PHP sendiri memiliki sifat Server-Side dikarenakan PHP dijalankan atau di eksekusi dari sisi server. Artinya, PHP dijalankan komputer server dan bukan pada komputer client. Sama seperti HTML, PHP pun dijalankan melalui aplikasi web browser.
PHP sendiri mulai dikembangkan pada tahun 1995 oleh Rasmus Lerdofr.

Fungsi dari PHP

Jika kita samakan dengan proses membuat rumah, pasti ada pondasinya dalam rumah tersebut, kita bisa menyebut HTML sebagai tiang-tiang pondasi, atap, lanai dan dinding. Pasti sangatlah kurang menarik dilihat rumah tersebut jika hanya mengandalkan bentuk / warna alami dari bahan material dasarnya saja, nah disini kita bisa meng-ibaratkan CSS untuk membuat bentuk dan tampilan rumah tersebut menjadi menarik dan enak di pandang. Pada contoh kasus bangunan ini PHP sendiri kita bisa ibaratkan sebagai yang bisa mengizinkan siapa saja yang dapat masuk ke dalam rumah, siapa yang dapat melihat rumah, dan siapa yang dapat menyimpan, memasukan atau membuang barang-barang yang berada pada rumah tersebut. Intinya PHP sendiri berguna untuk membuat aksi pada rumah atau website jika pada contoh nyatanya.

Penulisan Syntax PHP

Untuk penulisannya, PHP di mulai atau ditandai dengan membuat tag pembuka (<?php) dan di akhiri dengan tag penutup (?>). Pada syntax PHP sendiri kita bisa menyisipkan pada bagian-bagian HTML, kemudian diakhir setiap baris syntax PHP harus selalu di tutup dengan tanda semicolon atau titik koma (;), berikut contoh dalam penulisan PHP : 

<?php
  echo "Mengawali sebuah pembelajaran PHP di BERCODING";
?>

Anda bisa perhatikan pada contoh penulisan syntax PHP di atas, bahwa pada awal syntax PHP harus di buka dengan tag pembuka PHP yang ditandai dengan "<?php". Dan wajib ditutup dengan tag penutup PHP "?>". Dan jangan lupa perlu di perhatikan juga di akhir baris syntax PHP harus di akhiri atau di tutup dengan semicolon atau titik koma(;).
Syntax echo, berfungsi sebagai penampil data, adapun output yang akan di hasilkan pada contoh syntax PHP di atas ialah "Mengawali sebuah pembelajaran PHP di BERCODING".

Menjalankan PHP pada Localhost

Dalam menjalankan PHP pada Localhost, Anda harus memiliki/meng-install terlebih dahulu XAMPP sebagai alat yang membantu Anda untuk mempermudah proses pembelajaran PHP ini. Anda bisa mendownloadnya di sini Download XAMPP.

Setelah Anda menginstall XAMPP, yang harus Anda perhatikan ialah letak dari hasil instalasi-nya. Misalkan Anda menyimpan hasil instalasi pada direktori C:/xampp, berarti posisi dari web direktorinya berada pada C:/xampp/htdocs. Jika Anda ingin menyimpan selain di "C", misalkan Anda ingin menyimpan instalasi xampp di D:/xampp, berarti posisi dari web direktori berada pada D:/xampp/htdocs.

Setelah Anda mengetahui letak/posisi dari web direktorinya, langkah berikutnya adalah kita buat folder di dalam file htdocs. Semisal kita membuat folder dengan nama belajarPHP, jangan gunakan spasi dalam penulisan nama folder, tetapi anda bisa menggantinya dengan tanda underscore.

nantinya kita akan mengakses web direktori ini di browser dengan alamat url localhost, Anda bisa langsung menuliskan localhost, lalu web browser akan mengakses web direktori dalam contoh ini berada di c:/xampp/htdocs. Fungsi dari folder ini, supaya memudahkan kita dalam managemen file, kalau nanti di web direktori kita terdapat beberapa project aplikasi php yang kita buat.
Jadi kalau kita membuat folder dengan nama "belajarPHP", maka kita bisa mengakses folder tersebut dengan mengetikan url pada browser localhost/belajarPHP.
