String Pada PHP

Tipe data string adalah tipe data yang berisikan text, kalimat, atau kumpulan karakter. Diantaranya "a", "Hallo Apa Kabar", atau "S3lalu b3lajar y4H", itu semua bertipe data string. Tipe data ini mungkin tipe data yang paling sering digunakan, dan memiliki banyak fitur yang disediakan PHP. Karakter yang didukung saat ini adalah 256 Karakter ASCII..

Penulisan Tipe Data String pada PHP

PHP sendiri menyediakan 4 cara penulisan tipe data string, yakni Single Quoted, Double Quoted, Heredoc, dan Nowdoc.

Penulisan TIpe Data String dengan Single Quoted

Dalam penulisan Single Quoted, menggunkan tanda petik satu (karakter '...') merupakan cara penulisan string yang paling sederhana. Kita tinggal membuat sebuah kata atau kalimat, dan menambahkan tanda petik satu di awal dan akhir kalimat.
Untuk tipe data string yang didalamya juga terdapat tanda petik satu, kita harus mendahuluinya dengan karakter backslash(\) agar tidak dianggap sebagai penutup string. Dan jika di dalam string anda ingin menulis tanda backslash, kita harus menulisnya dengan 2 kali (\\).
Contoh penulisan Single Quoted : 

<?php
  $contoh1 = 'Ini adalah kalimat string contoh ke 1';

  $contoh2 = 'Sebuah string yang
  bisa memiliki beberapa baris';

  $contoh3 = 'Dia berkata : "Mengapa kau meninggalkanku?"';

  $contoh4 = 'Anda telah berhasil memasuki direktori C:\xampp\htdoc';

  $contoh5 = 'Kalimat ini tidak akan pindah ke : \n baris baru';

  $contoh6 = 'Variabel juga tidak otomatis ditampilkan $contoh1 dan $contoh3';

  echo $contoh1; echo "<br>";
  echo $contoh2; echo "<br>";
  echo $contoh3; echo "<br>";
  echo $contoh4; echo "<br>";
  echo $contoh5; echo "<br>";
  echo $contoh6;
?>

Escape String Pada PHP

Terkadang kita butuh mengetikkan karakter khusus seperti petik satu dan dua ( ' ' dan " "), atau tanda dollar $ di dalam string. Untuk itu, kita perlu menggunakan sesuatu yang disebut Escape Sequence atau Escape String. Berikut ini beberapa daftar Escape String yang dapat di pakai :

  • \$ unuk mengetikka tanda $ pada string petik dua,
  • \" untuk mengetikkan petik dua,
  • \' untuk mengetikkan petik satu,
  • \\ untuk mengetikkan back-slash \,
  • \n untuk newline (baris-baru/enter),
  • \r untuk carriage return,
  • \t untuk tab
contoh : 
<?php
  $nama = "Ramdani";
  echo "Isi variabel \$nama adalah \ "$nama\"";
?>

Variabel objek dan array dalam string

Objek dan Arrray pada PHP baru akan dibahas pada topik-topik berikutnya. Akan tetapi ada baiknya bila kita sedikit membahas objek dan array di sini terkait dengan string. String-petik-dua seperti yang dijelaskan dia ats, dapat juga berisi ogjek atau array, seperti pada contoh program berikut : 

<?php
  $objek = new stdClass();
  $objek -> nilai = 30;
  echo "Nilai pada objek : $objek -> nilai <br>";
  $perabots[] = "Kursi";
  $perabots[] = "Meja";
  $perabots[] = "Lemari";
  echo "Perabotan kita : $perabots[0], $perabots[1], $perabots[2] <br>";
  $harga['kursi'] = number_format(1500000,0,",",".");
  $harga['meja'] = number_format(500000,0,",",".");
  echo "Kursi : Rp. $harga[kursi], Meja : Rp. $harga[meja]";
?>
Pada contoh code tersebut, $objek merupakan suatu objek, $objek -> nilai merupakan field pada objek $objek. Field pada objek bisa dimasukkan langsung ke dalam String seperti contoh di atas. Akan tetapi field level dua seperti misalnya $objek -> format($angka). $perabots merupakan array berindex angka, dan $harga merupakan array ber-index string. Index array dapat diketik seperti biasa di dalam string. Tetapi untuk array ber-index string, nama index diketik tanpa tanda petik satu. Bila petik satu disertakan di dalam string untuk menampilkan variabel array, malah akan menyebabkan error. Perlu diketahui juga bahwa array dua tingkat juga tidak dapat dimasukkan ke dalam string. Bila array $koordinat[20][10] dimasukkan ke dalam string, tidak akan tampil seperti yang diharapkan.
