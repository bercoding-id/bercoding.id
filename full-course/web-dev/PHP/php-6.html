Operator Pada PHP

Di dalam bahasa pemograman sudah pasti ada operator, di PHP pun sama memiliki operator yang bisa untuk menyelesaikan tugas yang sesuai dengan penggunaannya. Semisal kita menggunakan percabangan pada PHP biasanya akan menggunakan operator pembanding, sedangkan bila kita menggunakan perulangan contoh for maka kita menggunakan operator penambahan atau pengurangan atau dalam bahasa inggris kita mengenalnya dengan Increment/Decrement. Dalam bahasa pemograman PHP memiliki beberapa jenis operator, diantaranya :

  1. Operator Aritmatika
  2. Operator Penugasan (Assignment)
  3. Operator Pembanding (Comparison)
  4. Operator Penambahan/pengurangan (Increment/Decrement)
  5. Operator String
  6. Operator Array

Operator Aritmatika

Dalam operator aritmatika digunakan untuk melakukan operasi aritmatika secara umum. Semisal penjumlahan, pengurangan, perkalian serta pembagian. Operasi aritmatika digunakan untuk tipe data numerik baik itu integer untuk bilangan bulat, maupun float untuk pecahan (bilangan desimal).

No Operator Keterangan
1 + Penjumlahan
2 - Pengurangan
3 * Perkalian
4 / Pembagian
5 % Modulus
6 ** Exponensial
Dalam operator tersebut kita bisa menggunakannya, sebagai contoh : 
<?php
  $x = 10;
  $y = 3;

  // Dengan menggunakan Penjumlahan
    $hasil = $x + $y;
    echo "Penjumlahan nilai $x + $y = $hasil";
    echo "<br>";

  // Dengan menggunakan Pengurangan
    $hasil = $x - $y;
    echo "Pengurangan nilai $x - $y = $hasil";
    echo "<br>";

  // Dengan menggunakan Perkalian
    $hasil = $x * $y;
    echo "Perkalian nilai $x * $y = $hasil";
    echo "<br>";

  // Dengan menggunakan Pembagian
    $hasil = $x / $y;
    echo "Pembagian nilai $x / $y = $hasil";
    echo "<br>";

  // Dengan menggunakan Sisa Bagi
    $hasil = $x % $y;
    echo "Modulus (nilai pembagian) nilai $x % $y = $hasil";
    echo "<br>";

  // Dengan menggunakan Pangkat
    $hasil = $x ** $y;
    echo "Exponensial (pangkat) nilai $x ** $y = $hasil";
?>

Operator Penugasan (Assignment)

Operator penugasan pada PHP bisanya digunakan untuk pemberian nilai pada Variabel. Dengan menggunakan simbol sama denga "=" memberi artian bahwa operan kiri diatur pada nilai expresi sebelah kanan.

No Operator Keterangan
1 = Pemberian Nilai
2 += Pemberian nilai dan penambahan
3 -= Pemberian nilai dan pengurangan
4 *= Pemberian nilai dan perkalian
5 /= Pemberian nilai dan pembagian
6 **= Pemberian nilai dan pemangkatan
7 %= Pemberian nilai dan modulus (sisa pembagian)
Sebagai contoh dari Operator Penugasan : 
<?php

  // Pengisian nilai ke variabel X
    $x = 20;

    echo "Nilai X = ".$x;
    echo "<br>";

  // Pengisian nilai ke variabel X dan penambahan
    $x = 20;
    $x += 100;

    echo "Penjumlahan nilai  X = ".$x;
    echo "<br>";

  // Pengisian nilai ke variabel X dan pengurangan
    $x = 50;
    $x -= 30;

    echo "Pengurangan nilai  X = ".$x;
    echo "<br>";

  // Pengisian nilai ke variabel X dan perkalian
    $x = 10;
    $y = 6;

    echo "Perkalian nilai  X & Y = ". $x * $y;
    echo "<br>";

  // Pengisian nilai ke variabel X dan pembagian
    $x = 10;
    $x /= 5;

    echo "pembagian nilai  X = ".$x;
    echo "<br>";

  // Pengisian nilai ke variabel X dan Modulus
    $x = 15;
    $x %= 4;

    echo "Modulus nilai  X = ".$x;
    echo "<br>";

  // Pengisian nilai ke variabel X dan Pemangkatan
    $x = 5;
    $x **= 4;

    echo "Pemangkatan nilai  X = ".$x;
    echo "<br>";

?>

Operator Pembanding

Operator pembanding adalah sebuah operator yang bisa membandingkan dua nilai. Nilai yang dibandingkan dapat berupa numerik ataupun string.

No Operator Keterangan
1 == Sama Dengan
2 === Nilai Yang Identik
3 != Tidak sama dengan
4 <> Tidak sama dengan
5 !== Tidak Identik
6 > Lebih besar dari
7 < Kurang dari
8 >= Lebih Besar dari sama dengan
9 <= Kurang dari sama dengan
Berikut ini contoh dari penggunaan dari Operator Pembanding : 
<?php

    $x = 100;  //deklarasi variabel x dengan nilai 100
    $y = 50;   //deklarasi variabel y dengan nilai 50

    var_dump($x == $y); // Nilai balik bernilai false, karena variabel x tidak sama dengan y

    echo"<br>";

    var_dump($x === $y); //Nilai balik bernilai false, karena tipe variabel x tidak sama dengan y

    echo"<br>";
    var_dump($x != $y); // Nilai balik bernilai true, karena variabel x tidak sama dengan y

    echo"<br>";
    var_dump($x <> $y); // Nilai balik bernilai true, karena variabel x tidak sama dengan y

    echo"<br>";
    var_dump($x !== $y); // Nilai balik bernilai true, karena variabel x tidak sama dengan y

    echo"<br>";
    var_dump($x > $y); // Nilai balik bernilai true, karena variabel x lebih besar dari y

    echo"<br>";
    var_dump($x < $y); // Nilai balik bernilai false, karena variabel x tidak lebih kecil dari variabel y

    echo"<br>";
    var_dump($x >= $y); // Nilai balik bernilai true, karena variabel x memang lebih besar dari y

    echo"<br>";
    var_dump($x <= $y); // Nilai balik bernilai false, karena variabel x tidak lebih kecil dari variabel y
?>

Operator penambahan/pengurangan

Fungsi dari operator ini adalah untuk menambahkan atau mengurangi sebuah nilai dari variabel. Dibawah ini beberapa operator yang digunakan

No Operator Keterangan
1 ++$x Pre-increment (Penambahan sebuah nilai pada variabel x, kemudian menjadi return pada nilai x tersebut)
2 $x++ Post-increment (Pengembalian nilai x, baru kemudian ditambahkan)
3 --$x Pre-decrement (Dikurangi sebuah nilai pada variabel c, kemudian di kembalikan ke nilai x)
4 $x-- Post-decrement (Dikembalikan nilai x, kemudian dikurangi sebuah nilai dari variabel x)
Sebagai contoh dari penggunaannya, seperti dibawah ini : 
<?php

    $x = 100;
    echo ++$x; //Menghasilkan nilai 101 karena menggunakan pre increment yang akan menambah nilai terlebih dahulu baru kemudian dicetak
    echo"<br>";

    $x = 100;  //Menghasilkan nilai 100 karena menggunakan post increment yang akan dicetak nilainya terlebih dahullu baru kemudian ada penambahan nilai
    echo $x++;
    echo"<br>";


    $x = 100;  //Menghasilkan nilai 99 karena menggunakan pre decrement yang akan mengurangi nilai terlebih dahulu baru kemudian dicetak
    echo --$x;
    echo"<br>";

    $x = 100;
    echo $x--; //Menghasilkan nilai 100 karena menggunakan pos decrement yang akan dicetak nilainya terlebih dahullu baru kemudian ada pengurangan nilai
?>

Operator Logika

Operator Logika pada umumnya digunakan untuk menggabungkan suatu kondisi tertentu.

No Operator Keterangan
1 and and (dan)
2 or or (atau)
3 xor xor (salah satu bernilai true maka hasilnya true)
4 && and (dan)
5 || or (atau)
6 ! not (tidak)
Sebagai contoh dari penggunaannya sebagai berikut : 
<?php

    $x = 100;
    $y = 50;

    if ($x == 100 and $y == 50) {
      echo "logika AND "; // kedua nilai semuanya harus bernilai true (benar)
    }

    echo "<br>";
    if ($x == 100 or $y == 80) {
      echo "logika OR"; // salah satu kondisi bernilai benar maka hasilnya true
    }

    echo "<br>";
    if ($x == 100 xor $y == 80) {
      echo "Logika XOR"; // salah satu kondisi bernilai benar maka hasilnya true, tetapi tidak untuk keduanya
    }

    echo "<br>";
    if ($x == 100 && $y == 50) {
      echo "Logika And (&&)";  // kedua nilai semuanya harus bernilai true (benar)
    }

    echo "<br>";
    if ($x == 100 || $y == 80) {
      echo "Logika OR (||)"; // salah satu kondisi bernilai benar maka hasilnya true
    }

    echo "<br>";
    if ($x !== 90) {
      echo "Logika Not"; // bernilai benar jika variabel x nilainya bukan 90
    }
?>
