Condition Pada PHP

Pengkondisian pada PHP sama saja dengan bahasa pemograman lainnya, seperti menggunakan kondisi IF ELSE. Kondisi IF ELSE bisa anda ibaratkan dengan pengecekan jjika maka pada kenyataannya. Fungsi dari IF ELSE sangat dibutuhkan pada saat Anda ingin memeriksa sebuah kondisi. Pada setiap logika terkadang kita memerlukan pengecekan suatu nilai yang jika terpenuhi untuk melakukan sesuatu dan jika pengecekan pertama tidak terpenuhi maka ada pengecekan kedua yaitu else. Dapat kita bayangkan tentang kondisi IF ELSE ini seperti misalnya anda memiliki seorang teman. Jika nama teman anda tersebut adalah Ramdani maka dia adalah laki-laki jika tidak maka dia adalah perempuan. Mungkin ada akan sedikit sulit jika di jelaskan. Akan sangat membantu Anda untuk memahami logika Kondisi IF ELSE ini menggunakan contoh dari code PHP. Berikut ini contoh dari penggunaan kondisi dari IF ELSE pada PHP. 

<?php
  if(kondisi){
    // Jika kondisi terpenuhi (Benar)
  }else{
    // Jika kondisi tidak terpenuhi (Salah)
  }
?>
Dalam pengaplikasiannya kedalam suatu kondisi sebagai berikut : 
<?php
  $friend = "Ramdani";

  if($friend == "Ramdani"){
    echo "Dia adalah seorang Programmer";
  }else{
    echo "Dia bukan seorang Programmer";
  }
?>
Maka hasilnya akan seperti ini :

Coba Anda perhatikan pada contoh yang barusan, variabel $friend berisi nilai string "Ramdani". Dan kemudian kita melakukan pengecekan pada variabel $friend, jika variabel $friend sama dengan "Ramdani", maka layar akan menampilkan "Dia adalah Seorang Programmer". jika tidak maka layar akan menampilkan "Dia bukan seorang Programmer"
Supaya Anda semakin paham kita akan mencoba pada kondisi yang salah, untuk contohnya sebagai berikut : 
  <?php
    $friend = "Ramdani";

    if($friend == "Arif"){
      echo "Dia adalah seorang Programmer";
    }else{
      echo "Dia bukan seorang Programmer";
    }
  ?>

Kondisi If-Elseif-Else

Kondisi Elseif dapat digunakan untuk memerika kondisi jika kondisi pertama (if) tidak terpenuhi. Sebagai contoh : 

<?php
  $friend = "Ramdani";

  if($friend == "Arif"){
    echo "Dia adalah seorang Programmer";
  }elseif($friend == "Ramdani"){
    echo "Dia adalah seorang Programmer";
  }else{
    echo "Dia bukan seorang Programmer";
  }
?>
Maka hasilnya akan seperti ini :

Variabel $friend berisikan nilai string "Ramdani" ,jika variabel $friend sama dengan "Arif" maka layar akan menampilkan "Dia adalah seoran Programmer", maka jika variabel $friend sama dengan "Ramdani" maka akan menampilkan "Dia adalah seorang Programmer". Jika tidak maka tampilkan "Dia bukan seorang Programmer", jadi disitu kondisi "elseif" digunakan jika Anda membutuhkan pengecekan kondisi kedua atau seterusnya jika kondisi pertama tidak terpenuhi maka kondisi kedua akan di cek, dan selanjutnya.
