Looping Pada PHP

Coba bayangkan jika Anda ingin mengulang suatu hal, Sebagai berikut : 

<?php
  echo "<h2>Ini adalah sebuah Kalimat Contoh </h2>";
  echo "<h2>Ini adalah sebuah Kalimat Contoh </h2>";
  echo "<h2>Ini adalah sebuah Kalimat Contoh </h2>";
  echo "<h2>Ini adalah sebuah Kalimat Contoh </h2>";
  echo "<h2>Ini adalah sebuah Kalimat Contoh </h2>";
?>
Dengan cara itupun Anda masih bisa melakukan pengulangan tetapi jika Anda ingin mengulanginya 100 kali, pasti akan kesulitan bukan? dan pasti akan memakan waktu lama untuk mengerjakannya. Karena itu kita harus menggunakan perulangan (Looping). Ada dua jenis perulangan dalam pemrograman, diantaranya :
  1. Counted Loop
  2. Uncounted Loop
Perbedaan dari keduanya ialah, Counted Loop adalah perulangan yang sudah jelas banyak pengulangnya. Sedangkan Uncounted Loop tidak pasti berapa kali dia akan mengulang.
Pada bahasa pemrograman PHP, ada 4 jenis perulangan yang bisa Anda gunakan, sebagai berikut :
  1. Perulangan For
  2. Perulangan While
  3. Perulangan Do/While
  4. Perulangan Foreach
Supaya Anda makin paham cara penggunaan dan sifat dari masing-masing perulangan tersebut, kita akan bahas satu persatu.

Perulangan For

Perulangan For adalah perulangan yang termasuk dalam Counted Loop, karena kita bisa menentukan jumlah perulangannya. Penulisan perulangan For sebagai berikut : 

<?php
  for ($i = 0; $i < 10; $i++){
    // Blok kode yang akan diulang di sini!
  }
?>
Dalam perulangan For, Variabel $i berfungsi sebagai counter yang menghitung berapa kali ia akan mengulang. Hitungan akan dimulai dari nol (0), karena memberikan nilai $i = 0. Lalu, perulangan akan diulang selama nilai $i lebih kecil dari 10. Yang berarti, perulangan ini akan mengulang sebanyak 10x. Makhsud dari $i++ adalah nilai $i akan ditambah 1 disetiap kali melakukan perulangan.
Supaya Anda makin memahami cara dari perulangan For, dibawah ini contoh dari penggunaan perulangan For : 
<?php
  for ($i = 0; $i < 10; $i++){
    echo "<h2>Ini kalimat perulangan ke-$i</h2>";
  }
?>

Perulangan Do/While

Perulangan While adalah perulangan yang termasuk ke dalam Uncounted Loop. Karena pada umumnya digunakan untuk mengulang sesuatu yang belum jelas jumlah perulangannya. Namun, perulangan While juga bisa digunakan seperti perulangan for sebagai counted loop. Penulisan perulangan Do/While, sebagai berikut :

<?php
  white(<kondisi>){
    // Blok kode yang akan diulan disini!
  }
?>
Contoh dalam kondisi sebenarnya, sebagai berikut : 
<?php

  $ulangi = 0;

  while($ulangi < 10){
    echo "<p>Ini adalah perulangan ke-$ulangi</p>";
    $ulangi++;
  }
?>

Dalam perulangan while akan terus mengulang selama nilai dari variabel $ulangi lebih kecil dari 10. Lalu di dalam perulangan kita melakukan increment nilai variabel $ulangi dengan $ulangi++. yang berarti di setiap pengulangan akan bertambah 1.

Perulangan Do/While

Pada perulangan Do/While sama seperti perulangan while/ Ia juga termasuk kedalam Uncounted Loop. Perbedaan dari Do/While dengan while terletak pada cara memulai pengulangannya. Perulangan Do/While akan selalu melakukan pengulangan sebanyak 1 kali, kemudian melakukan pengecekan kondisi. Sedangkan perulangan while akan mengecek kondisinya terlebih dahulu, baru akan melakukan pengulangan.
Bentuk dasar dari perulangan Do/While : 

<?php
  do {
    // blok kode yang akan diulang
  }while (<kondisi>);
?>
Contoh dalam kondisi nyata, sebagai berikut : 
<?php
  $ulangi = 10;

do {
  echo "<p>Perulangan yang ke-$ulangi</p>";
  $ulangi--;
  } while ($ulangi > 0);

?>

Perulangan Foreach

Perulangan Foreach sama seperi perulangan for. Tetapi, perulangan Foreach lebih khusus digunakan untuk mencetak array. Berikut bentuk dari perulangan foreach : 

$lt;?php
  foreach($array as $data){
    echoo $data;
  }
?>
Agar Anda semakin paham cara penulisan dari perulangan foreach, dibawah ini contoh dari perulangan foreach ketika sudah dimasukan kondisi tertentu : 
<?php
  $books = [
            "Panduan Belajar PHP untuk Pemula",
            "Membangun Aplikasi Web dengan PHP",
            "Tutorial PHP dan MySQL",
            "Membuat Chat Bot dengan PHP"
  ];

  echo "<h5>Judul Buku PHP:</h5>";
  echo "<ul>";
  foreach($books as $buku){
    echo "<li>$buku</li>";
  }
  echo "</ul>";

?>
