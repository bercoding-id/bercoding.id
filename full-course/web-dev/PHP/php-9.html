Fungsi Pada PHP

Seperti pada bahasa-bahasa pemograman lainnya, di PHP pun ada fungsi (function) yang bisa digunakan untuk mempermudah dalam mengerjakan suatu project. Banyak fungsi build-in dari php yang sering kita gunakan, seperti print(), print_r(), unset(), dll. Selain fungsi-fungsi tersebut, kita juga dapat membuat fungsi sendiri sesuai kebutuhan kita. Fungsi adalah sekumpulan instruksi yang dibungkus dalam sebuah blok. Fungsi dapat digunakan ulang tanpa harus menulis ulang instruksi di dalamnya. Fungsi pada PHP dapat dibuat dengan kata kunci function, lalu diikuti dengan nama fungsinya. Sebagai berikut : 

<?php
  function namaFungsi(){
    //...
  }
?>
kode instruksi dapat anda tulis di dalam kurung kurawal ({...}). Sebagai contoh : 
<?php
  function perkenalan(){
    echo "Assalamualaikum, kawan<br>";
    echo "Nama saya Ramdani<br>";
    echo "Senag bisa mengenal kamu<br>"
  }
Fungsi yang sudah kita buat tidak akan menghasilkan apapun jika kita tidak memanggilnya. Kita dapat memanggil fungsi dengan menuliskan namanya. Sebagai contoh : 
<?php
  // Membuat suatu fungsi
  function perkenalan(){
    echo "Assalamualaikum, kawan";
    echo "Perkenalkan saya Ramdani<br>";
    echo "Senang berkenalan dengan kamu<br>";
  }
  // Memanggil fungsi yang sudah dibuat
  perkenalan();

  echo "<hr>";

  // Memanggil lagi
  perkenalan();
?>

Fungsi dengan Parameter

Supaya intruksi yang di dalam fungsi lebih dinamis lagi, kita dapat menggunakan parameter untuk memasukkan sebuah nilai ke dalam fungsi. Nilai tersebut akan diolah di dalam fungsi. Misalkan, pada contoh fungsi yang tadi kita buat, tidak mungkin nama yang dicetak adalah Ramdani saja dan salam yang dipakai tidak selalu Assalamualaikum. Maka, kita dapat menambahkan parameter menjadi seperti ini : 

<?php
  // Membuat fungsi
  function perkenalan($nama, $salam){
    echo $salam. ",<br>" ;
    echo "Perkenalkan, nama saya ".$nama." <br>";
    echo "Senang berkenalan dengan anda<br>";
  }

  // Memanggil fungsi yang sudah dibuat
    perkenalan("Linda", "Hi");

    echo "<hr>";

    $saya = "Ramdani";
    $ucapanSalam = "Selamat Pagi";

    // Memanggil lagi
    perkenalan($saya, $ucapanSalam);
  ?>
Maka hasilnya akan seperti ini :

Fungsi dengan Nilai Default

Nilai default dapat kita berikan di parameter. Nilai default berfungsi untuk mengisi nilai sebuah parameter, kalau parameter tersebut tidak diisi nilainya. Misalnya : saya lupa mengisi parameter nama, maka program akan error. Oleh karena itu, kita perlu memberika nilai default supaya tidak terjadi error. Sebagai contoh : 

<?php
  // Membuat fungsi
  function perkenalan ($nama="Ramdani", $salam){
    echo $salam.",<br>";
    echo "Perkenalkan, nama saya ".$nama."<br>";
    echo "Senang beejumpa dengan kamu "<br>"";
  }

  // Memanggil fungsi yang sudah dibuat
  perkenalan("Intan", "Hi");

  echo "<hr>";

  $saya = "Ramdani";
  $ucapanSalam = "Selamat Siang";

  // Memanggilnya lagi tanpa mengisi parameter nama
  perkenalan($ucapanSalam);
?>
Maka hasilnya akan seperti ini :

Fungsi yang Mengembalikan Nilai

Hasil pengolahan nilai dari fungsi mungkin saja kita butuhkan untuk pemrosesan berikutnya. Oleh karena itu, kita harus membuat fungsi yang dapat mengembalikan nilai. Pengembalian nilai dalam fungsi dapat menggunakan kata kunci return. Sebagai contoh : 

<?php
  // membuat suatu fungsi
  function hitungUmur($thn_lahir, $thn_sekarang){
    $umur = $thn_sekarang - $thn_lahir;
    return $umur;
  }

  echo "Umur saya sudah menginjak ". hitungUmur(1993, 2020) . " Tahun";
?>

Memanggil Fungsi di dalam Fungsi

Fungsi yang sudah kita buat, dapat juga kita panggil di dalam fungsi laim. Sebagai contoh : 

<?php
  // Membuat fungsi
  function hitungUmur($thn_lahir, $thn_sekarang){
    $umur = $thn_lahir - $thn_sekarang;
    return $umur;
  }

  function perkenalan($nama, $salam="Selamat Pagi!"){
    echo $salam. ",<br>";
    echo "Perkenalkan, Nama saya ".$nama."<br>";

    // Memanggil fungsi lain
    echo "Saya berusia ". hitungUmur(1993, 2018) ." Tahun<br>";
    echo "Senang berkenalan dengan Anda<br>";
  }

  // Memanggil fungsi perkenalan
  perkenalan("Ramdani");

?>
Maka hasilnya akan seperti ini :
