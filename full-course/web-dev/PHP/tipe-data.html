Tipe Data PHP

Sama halnya bahasa pemrograman lainnya, PHP juga memiliki beberapa tipe data. Tipe data bisa dikatan sebagai tipe dari isi variabel. Ada yang berbentuk angka yang biasa disebut dengan tipe data integer, dan adapula yang berbentuk kalimat atau kata, yang biasa di sebut dengan tipe data string. Beriktu ini ialah beberapa tipe data yang berada pada PHP :

  1. String
  2. Integer
  3. Float
  4. Boolean
  5. Array
  6. Object
  7. NULL
dari tipe data PHP di atas, hampir semua tipe data tersebutt sangat sering digunakan dalam proses pembangunan sebuah aplikasi.

Tipe Data String

Tipe data string ialah tipe data yang berbentuk text, dan untuk penulisannya tipe data string ini diletakkan di tengah-tengah tanda petik. Yang diawali dengan tanda petik dan di akhiri dengan tanda petik juga. Berikut ini contoh dari penulisan tipe data string : 

<?php
  $contoh = "Saya Sedang Belajar";
  // variabel contoh di atas memiliki tipe data string dikarenakan berisi text atau kalimat.
?>
Pada contoh code diatas sebuah variabel dengan nama contoh yang berisi kalimat "Saya Sedang belajar", memiliki tipe data string. dikarenakan sebuah nilainya berada di dalam tanda petik dua ("...").

Tipe Data Integer

Tipe data integer ialah sebuah tipe data yang berbentuk angka yang berbentuk/bernilai bilangan asli atau bilangan bulat, untuk penulisan tipe data tersebut tidak perlu menambahkan tanda petik dua seperti tipe data string
Berikut contoh penulisannya : 

<?php
  $bilangan1 = 10;
  $bilangan2 = 73;

// Kedua variabel tersebut merupakan variabel yang beripe data integer
// dikarenakan nilanya bilangan asli atau bulat.
?>
Untuk cara penulisan variabel yang beripe data integer, Anda tidak memerluka tanda petik atau apapun, dengan cuman mengisikan sebuah nilai bilangan asli dan bulat. Nilai tersebut sudah bertipe data integer.

Tipe Data Float

Tipe data float atau bisa di sebut juga tipe data double ialah tipe data yang berisikan nila bilangan desimal. Cara penulisannya hampir sama dengan tipe data integer karena tidak memerluka tanda petik di awal dan di akhir isi variabel float. Berikut contoh dalam penulisannya : 

<?php
  $angka = 13.232;
// variabel tersebut bertipe data float dikarenakan nilanya bilangan desimal
?>

Tipe Data Boolean

Untuk tipe data boolean sendiri, hanya memiliki duan buah value atau isi, yaitu true dan false, atau 0 dan 1. True sendiri sama saja dengan 1, dan false sendiri sama saja dengan 0. Tipe data ini sering digunakan untuk memeriksa ketersediaan atau memerika nilai kebenara dari sebuah data.
Berikut contoh penulisan tipe data boolean : 

<?php
  $nilai1 = true;
  $nilai2 = false;
// variabel nilai1 & nilai2 memiliki tipe data boolean karena
// hanya berisikan nila benar dan salah saja.
?>
Setelah anda mengetahui cara penulisan dari tipe data pada PHP, di pembahasan selanjutnya Anda akan lebih memahami kegunaan dan sifat dari variabel sendiri.
