Pengenalan Sass

Apa Itu Sass?

Sass adalah bahasa scripting yang memudahkan Anda untuk menulis CSS dengan lebih mudah dan efisien. Ada 2 cara untuk menulis Sass, sintaksis SASS dan sintaksis SCSS. Di , kita menggunakan sintaksis SCSS, yang mana adalah sintaksis yang lebih umum. File ekstensinya adalah .scss dan bukan .css.

Environment Sass

Sebelum menjalankan file sass, kita harus menginstall terlebih dahulu sass melalui terminal cmd.
Pertama, terlebih dahulu meng-install ruby, bisa didownload installer-nya disini.
Kedua, buka command prompt.
Lalu, install dengan mengetikan "gem install sass" di cmd.
Untuk memeriksa apakah sass sudah ter-install, ketikkan "sass -v" di cmd.

Manfaat Sass

Sass memiliki manfaat, antara lain :

  • Lebih singkat dari CSS
  • Kode dapat digunakan ulang

Struktur Nesting (Sarang) dalam Sass

Mari kita lihat struktur nesting yang sering digunakan dalam Sass. Dengan nesting, kita dapat menulis ulang CSS seperti contoh dibawah ini.

  • index.html
  • style.css
  • style.scss

Contoh file HTML

<html>
    <head>
        <title> :Active </title>
        <link rel="stylesheet"href="style.css">
    </head>
    <body>
        <div class="header">
            <ul>
                <p>Example</p>
            </ul>
        </div>
    </body>
</html>

Contoh file CSS

.header {
    width: 100%;
}
.header ul {
    padding: 10px;
}

Contoh file SCSS

.header {
    width: 100%;
    ul {
        padding: 10px;
    }
}
Dengan cara ini, Sass memungkinkan kita untuk nesting (menyarang) selector pada selector lain. Jadi, dengan Sass, kita tidak perlu untuk menulis selector yang sama berulang kali.

Keuntungan Nesting (Sarang)

Semakin besar site, makin berguna nesting. Terutama ketika melakukan perubahan pada nama class. Jika kita ingin mengubah nama class header pada contoh di bawah, kita harus melakukan perubahan di 3 lokasi dengan CSS normal, namun dengan Sass kita hanya perlu melakukan satu perubahan.

  • style.css
  • style.scss

Contoh file CSS

.header {
    width: 100%;
}
.header ul {
    padding: 10px;
}
.header ul li {
    font-size: 15px;
}

Contoh file SCSS

.header  {
    width: 100%;
    ul {
        padding: 10px;
        li {
            font-size: 15px;
        }
    }
}
