Simbol & (1)

Mengulas kembali tentang :hover

Sebelum kita mempelajari tentang simbol &, mari kita kembali mengulas ulang tentang :hover dan :active. Yaitu, sebuah keadaan dimana kursor berada diatas element disebut :hover. Pada CSS, dengan menggunakan selector :hover, kita dapat menerapkan CSS spesifik saat kursor menghover sebuah element.

  • index.html
  • style.css
  • Output

Berkas HTML

<html>
    <head>
        <title> :Hover </title>
        <link rel="stylesheet"href="style.css">
    </head>
    <body>
        <div>
            <h1>Tes Hover</h1>
        </div>
    </body>
</html>

Berkas CSS

div:hover {
    background-color: blue;
}

Hasil dari selector :hover

Mengulas kembali tentang :active

Dengan menggunakan :active, CSS dapat diterapkan hanya jika element diklik. Seperti contoh dibawah ini :

  • index.html
  • style.css
  • Output

Berkas HTML

<html>
    <head>
        <title> :Active </title>
        <link rel="stylesheet"href="style.css">
    </head>
    <body>
        <div>
            <button>Klik Tombol</button>
        </div>
    </body>
</html>

Berkas CSS

button {
    margin-top: 50px;
    display: flex;
    justify-content: center;
}
button {
    padding: 20px 50px;
    font-size: 20px;
    box-shadow: 0 9px black;
    border-radius: 10px;
}
button:active {
    position: relative;
    top: 7px;
    box-shadow: none;
}

Output

Simbol &

Dengan menggunakan nesting (sarang), kita dapat menulis ulang selector :hover menjadi &:hover. Simbol & dapat digunakan dengan selectro apapun -hover, -active, dan lain-lain. Seperti contoh dibawah ini :

  • style.css
  • style.scss

Berkas CSS

li {
    font-size: 15px;
}
li:hover {
    background-color: blue;
}

Berkas SCSS

li {
    font-size: 15px;
    &:hover {
        background-color: blue;
    }
}
