Simbol & (2)

Menerapkan CSS ke dalam element spesifik

Saat kita ingin menerapkan CSS ke tag <li> spesifik seperti pada contoh di bawah, kita dapat menggunakan sintaksis element.class-name. Sintaksis ini sering digunakan untuk menerapkan CSS ke element spesifik diantara banyak lainnya.

  • index.html
  • style.css
  • Output

Berkas HTML

<html>
    <head>
        <title> :Active </title>
        <link rel="stylesheet"href="style.css">
    </head>
    <body>
        <div class="header">
            <ul>
                <li>Satu</li>
                <li>class="second"Dua</li>
                <li>Tiga</li>
            </ul>
        </div>
    </body>
</html>

Berkas CSS

li {
    font-size: 20px;
}
li.second {
    color: blue;
}

Mereferensikan element spesifik

Pada halaman sebelumnya, kita sudah belajar untuk mereferensikan element spesifik dengan mudah dengan menggunakan simbol & pada Sass. Dengan CSS normal, kita harus menulis nama element yang sama berulang kali, namun pada Sass kita dapat menulisnya sekali sehingga mudah untuk mengaplikasikan kode CSS berbeda ke element spesifik.

  • index.html
  • style.css
  • style.scss

Berkas HTML

<html>
    <head>
        <title> :Active </title>
        <link rel="stylesheet"href="style.css">
    </head>
    <body>
        <div class="header">
            <ul>
                <li>Satu</li>
                <li>class="second"Dua</li>
                <li>class="third"Tiga</li>
            </ul>
        </div>
    </body>
</html>

Berkas CSS

li {
    font-size: 20px;
}
li.second {
    color: blue;
}

Berkas SCSS

li {
    font-size: 20px;
    &.second {
    color: blue;
    }
    &.third {

    }
}
