Variable Sass

Variable

Selanjutnya, kita akan belajar mengenai variable pada Sass. Bayangkan variable sebagai kotak yang mempunyai nilai. Gambar di bawah mengilustrasikan bagaimana variable bekerja. Mereka berguna pada kasus dimana nilai yang sama dipakai berulang kali.

Mendefinisikan Variable

Untuk menggunakan variable, kita harus terlebih dahulu mendefinisikan mereka. Variable didefinisikan dengan sintaksis: $variable-name: nilai; seperti ditunjukkan di bawah. Kita dapat memikirkannya seperti "letakkan nilai di kanan ke variable di kiri", ini disebut assignment.

  • Cara Kerja
  • style.scss

Cara Kerja


Berkas SCSS

// $bercoding-color adalah sebuah nama variabel                                            
// #126e82 adalah sebuah nilai dari variabel                                            
// #126e82 ditetapkan menjadi nilai di dalam variabel $bercoding-color                                            
$bercoding-color:  #126e82;

$bercoding-color adalah sebuah nama variabel

#126e82 adalah sebuah nilai variabel


Berkas SCSS

li {
    font-size: 20px;
    &.second {
    color: blue;
    }
    &.third {

    }
}

Cara Menggunakan Variable Sass

Menggunakan variable itu mudah. Cukup tulis nama variable, contohnya $bercoding-color, seperti terlihat di bawah. Dengan menulis nama variable di tempat dimana Anda ingin menggunakannya, value #126e82 yang tersimpan di dalam variable akan otomatis disisipkan. Ini memungkinkan Anda untuk mereferensikan value yang sama dari tempat berbeda, membuatnya mudah untuk diubah.

  • style.scss

Berkas SCSS

// color dalam tag p dan h1 menggunakan nilai #126e82 dari variabel $bercoding-color dari pendefinisian                                                                                     
$bercoding-color:  #126e82; 
h1 {
    color: $bercoding-color;
}
p {
    color: $bercoding-color;
}

Penempatan Definisi Variable Sass

Variable harus didefinisikan sebelum baris dimana ia akan digunakan. Karena style sheet (CSS) dibaca dari atas ke bawah, jika kita mendefinisikan sebuah variable setelah baris dimana kita ingin menggunakannya, error akan terjadi karena itu akan diinterpretasikan sebagai variable yang tidak terdefinisi.

  • style.scss

Berkas SCSS

<!--- dapat digunakan --->                                            
$text-red: #ff000;
h1 {
    color: $text-red; //dapat digunakan
    background-color: $back-blue; //tidak dapat digunakan sebelum didefinisikan
}
<!--- tidak dapat digunakan sebelum didefinisikan --->  
$back-blue: #0000ff;

Scope Variable Sass

Kita juga dapat menspesifikasikan jarak variabel. Jarak ini disebut dengan scope. Kita tidak bisa menggunakn variabel jika variabel tidak ada didalam scope-nya. Maka dari itu, semua variabel yang ingin digunakan harus didefinisikan dibagian terluar dari struktur nesting (sarang).

  • style.scss

Berkas SCSS

h1 {
    $text: #126e82;
    color: $text; //dapat digunakan karena sudah didefinisikan di h1
}
p {
    color: $text; //tidak dapat digunakan karena berada diluar scope
}
