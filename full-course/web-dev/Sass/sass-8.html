Import

Import Sass

Mari mempelajari bagaimana memuat file-file eksternal dengan import. Agar suatu file dapat diimport, kita harus menambahkan _ diawal nama file. Selain itu, sintaksis umum untuk memuat file adalah @import filename seperti pada contoh di bawah ini.

  • _colors.scss
  • style.scss

Berkas _colors.scss yang ditambahkan garis bawah pada awal nama file

$bercoding-color: #126e82;

Berkas SCSS untuk meng-import berkas _colors.scss

@import"_colors.scss"; //unduh file
p {
    color:$bercoding-color;
    font-size: 15px;
} 
//@import unduh file "_colors.scss", sehingga dapat menggunakan variable $bercoding-color

Mempersingkat Nama File

Perlu diingat bahwa saat mengimport file, kita dapat menghapus _ dan ekstensi file (.scss) seperti pada contoh di bawah ini.

  • _variable.scss
  • style.scss

Berkas _variable.scss

$html: #0ac4ff;
$javascript: #126e82;

Berkas SCSS untuk meng-impot berkas _variable.scss

@import"variable" //garis bawah dan ekstensi file dapat dihapus
.html {
    background-color: $html;
}
.javascript {
    background-color: $javascript;
}
