Tag HTML

Tag dalam HTML

Tag HTML digunakan untuk menandai elemen dalam suatu dokumen HTML. Tag HTML terdiri atas sebuah sudut kiri ( <, tanda lebih kecil), nama sebuah tag, dan sebuah tanda kurung sudut kanan ( >, tanda lebih besar). Tag umumnya berpasangan, sebagai contoh < h1 > dengan < / h1 >. Secara umum suatu elemen dalam dokumen HTML yang dinyatakan dengan tagnya, dituliskan seperti ini : < namatag> - < / namatag>
